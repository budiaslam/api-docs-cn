# 生成直接支付接口

**生成直接支付 API** 可通过多种渠道实现无缝支付处理，包括 **QRIS 支付**、**银行账户转账** 和 **虚拟账户（VA）支付**。\
该 API 通过允许商户直接发起支付来简化交易流程。

**支持的支付方式：**

* **QRIS 支付** — 通过二维码处理支付
* **银行账户转账** — 通过银行账户直接转账以确保交易安全。
* **虚拟账户（VA）支付** — 通过虚拟账户自动完成支付。
