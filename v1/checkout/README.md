# 结账流程

我们的 **Checkout API** 提供流畅的基于发票的支付流程，让客户能够安全地完成交易

**主要功能：**

* **即时发票生成** — 系统会为每笔交易自动创建详细发票
* **灵活的支付方式** — 客户可使用其首选的支付方式（如信用卡/借记卡、银行转账或其他支持的选项）
* **安全的交易处理** — 所有交易均经过加密，并符合行业安全标准
* **支付状态追踪** — 商户与客户可实时监控支付状态。

商户可通过 **API** 轻松管理并对账交易记录。
